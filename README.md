# 🎓 Campus ERP — Proof of Concept

A modern, role-based College ERP system built with React + Vite. Designed as a PoC for colleges.

🔗 **Live Demo:** [sashn108.github.io/college-erp-poc](https://sashn108.github.io/college-erp-poc/)

---

## 🚀 Features

### 👨‍🎓 Student Portal
| Module | Features |
|---|---|
| Dashboard | Stats, attendance bars, recent notices |
| Attendance & Leave | Subject-wise %, leave apply & history |
| Question Papers | Semester-wise previous year papers |
| Assignments | Upload, track status, view scores |
| Exams & Results | Schedule, marksheet, SGPA |
| Fee & Finance | Fee structure, hostel, scholarship |
| Notices | Category-tagged announcements |

### 👨‍🏫 Faculty Portal
| Module | Features |
|---|---|
| Dashboard | Today's schedule, pending actions |
| Subjects & Students | Assigned subjects, student list, direct mail |
| Lab Management | Session tracker, file submission counts |
| Attendance & Leave | Mark/view attendance, leave approval |
| Evaluation | Assignment & project grading queue |
| Research Scholars | Progress tracking, milestone management |
| Exam & Duty | Invigilation & paper-setting roster |

## 🛠️ Tech Stack
- **React 18** — UI
- **Vite 5** — Build tool
- **Pure CSS-in-JS** — No external UI library
- **GitHub Pages** — Hosting

---

## 📦 Run Locally

```bash
npm install
npm run dev
```

## 🚢 Deploy to GitHub Pages

```bash
npm run build
# then push the dist/ folder to gh-pages branch
```
